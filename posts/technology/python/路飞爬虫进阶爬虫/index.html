视频网站工作原理#

  1. 用户上传视频到视频网站

  2. 视频网站对视频进行转码(4K,2K,1080P,标清等)

  3. 对视频进行切片处理（个人理解不知道对错，分布式存储视频片段） ==================被切分的视频片段，当用户想拖到后面的片段播放时就直接播放后面被切分的一小段片段，节省流量资源

  4. 切分后的视频的排序问题，需要一个文件1.记录视频播放顺序2.视频的存放路径基本上会直接存放在M3U文件(固定格式的文本)如json，txt，M3U文件通过utf-8编码后的名字是M3U8

    怎么爬取？

    1. 找到m3u8文件（通过各种手段）
    2. 通过m3u8下载视频文件
    3. 将下载到的视频文件合并成一个视频文件

selenium的视频学习#

tips：

  1. 所打开的浏览器的左上角的浏览器受到控制提示可能后面会遇到反爬
  2. 如果遇到ajax的页面(局部刷新)所要操作时需要等待否则大概率报错
  3. 当代码只爬取到了部分数据就报错了一般是爬取太快所导致
  4. 有些视频网站会有iframe，必须先要拿到iframe(iframe = find_element()找到iframe)然后把视角切换到iframe(web.switch_to.window(iframe窗口))才能拿到数据，切换回原页面使用web.switch_to.default_content()

先粗细定位整体在精细定位局部

  1. 如果你的程序chrome被检测到了是爬虫程序，一般分为两种情况解决chrome版本号小于88和大于88（浏览器控制台输入window.navigator.webdriver查看）

import time
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.select import Select
#无头浏览器代码
chrome_opt = webdriver.ChromeOptions()
chrome_opt.add_argument('--headless')
browser = webdriver.Chrome(options=chrome_opt)

url = 'https://www.endata.com.cn/BoxOffice/BO/Year/index.html'
browser.get(url)

sel_el = browser.find_element(By.XPATH, value='//*[@id="OptionDate"]')  #选择下拉框的提取
sel = Select(sel_el)    #将下拉框包装成select对象

#让浏览器调整选项
for i in range(len(sel.options)):
    sel.select_by_index(i)  #拿到每一个下拉框的选项
    time.sleep(1)
    table = browser.find_element(By.XPATH, value='//*[@id="TableList"]')
    print(table.text)
    time.sleep(1)

import time
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.common.keys import Keys
url = "https://www.lagou.com/"
browser = webdriver.Chrome()
browser.get(url)
time.sleep(1)
browser.find_element(By.XPATH,'//*[@id="changeCityBox"]/p[1]/a').click()
time.sleep(1)
browser.find_element(By.XPATH,'//*[@id="search_input"]').send_keys("爬虫", Keys.ENTER)
time.sleep(1)

#接下来提取公司名称，岗位名称，工作地点
# 查找存放数据的位置，进行数据提取
#进行列表全部数据的粗提取
all_list = browser.find_elements(By.XPATH, value='//*[@id="jobList"]/div[1]/div')

for li in all_list:
    li.find_element(By.XPATH, value='./div[1]/div[1]/div[1]/a').click()
#                                             //*[@id="jobList"]/div[1]/div[1]/div[1]/div[1]/div[1]/a
    time.sleep(1)
    browser.switch_to.window(browser.window_handles[-1])    #切换窗口，将它置为最后一个窗口
    time.sleep(1)
    datajob = browser.find_element(By.XPATH, value='//*[@id="job_detail"]/dd[2]').text
    print(datajob)
    browser.close()
    browser.switch_to.window(browser.window_handles[0])
    time.sleep(1)

# coding:utf-8
# @Time : 2022/5/21 11:41
# @Author : 软件1194温铭军
# @file : 12306.py
# $software : PyCharm
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.common.keys import Keys
import time
from selenium.webdriver.common.action_chains import  ActionChains #鼠标操作导包,事件链
#防止被认出是爬虫程序，chrome版本88以上适用
chrome_opt = webdriver.ChromeOptions()
chrome_opt.add_argument('--disable-blink-features=AutomationControlled')
browser = webdriver.Chrome(options=chrome_opt)

url = 'https://kyfw.12306.cn/otn/resources/login.html'
browser.get(url)
time.sleep(1)
#输入账号密码
browser.find_element(By.XPATH, value='//*[@id="J-userName"]').send_keys('你的账号')
browser.find_element(By.XPATH, value='//*[@id="J-password"]').send_keys('密码')
browser.find_element(By.XPATH, value='//*[@id="J-login"]').click()
time.sleep(2)

#拖拽目标按钮
btn = browser.find_element(By.XPATH, value='//*[@id="nc_1_n1z"]')

#找到目标拖拽验证码
browser.find_element(By.XPATH, value='//*[@id="nc_1__scale_text"]/span')


#拖拽移动
ActionChains(browser).drag_and_drop_by_offset(btn, 300, 0).perform()  #横推拽
