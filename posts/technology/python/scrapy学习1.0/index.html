Scrapy学习#

使用Scrapy创建爬虫项目使用如下步骤：#

  1. 创建爬虫项目scrapy startproject 项目名字(项目名字不允许使用数字开头，不能包含中文)
  2. 创建爬虫文件，在spiders文件夹中创建爬虫文件
    1. 先进入到要创建爬虫文件的目录cd 项目名字\项目名字\spiders
    2. 创建爬虫文件scrapy genspider 爬虫文件的名字 要爬取的网页（要爬取的网页一般不写http只写www）
  3. 运行爬虫代码scrapy crawl 爬虫名字

scrapy项目结构#

项目名字

​ 项目名字

​ spiders文件（存储的是爬虫文件）

​ init

​ 自定义的爬虫文件 核心功能文件

​ init

​ itmes 定义数据结构的地方爬取的数据包含哪些，是一个继承自scrapy.Item的类，通俗的说要下载哪些数据

​ middlewares 中间件 代理

​ pipelines 管道 里面只有一个类用来处理下载的数据，默认值是300优先级，值越小优先级越高（1-1000）

​ settings 配置文件robots协议、UA等配置

response的属性和方法#

response.text 获取的是响应字符串 response.body 获取的是二进制数据 response.xpath 可以直接通过xpath方法来解析response中的内容

response.extract() 提取seletor对象的data属性值，现在一般使用get()

response.extract_first() 提取seletor列表的第一个数据下面，现在一般使用getall()

管道有优先级，值越小优先级越高

  1. 现在pipelines中定义自己的功能管道

  3. 写管道功能

settings数据库配置#

  1. 导入settingsfrom scrapy.utils.project import get_project_settings
  2. 在管道文件中配置数据库连接参数
    1. settings = get_project_settings()
    2. self.host = settings['DB_HOST']，参考在settings中的配置等等。

scrapy架构#

  1. 引擎
  2. 下载器
  3. spiders
  4. 调度器
  5. 管道

官网架构概述图如下：

自己总结版如下：

  1. 引擎向spiders要url
  2. 引擎在得到url后，将要爬取的url给调度器
  3. 调度器将url生成对象放入指定的队列中
  4. 调度器从队列中出队一个请求
  5. 引擎将请求交给下载器进行处理
  6. 下载器发送请求到互联网获取数据
  7. 下载器将response数据返回给引擎
  8. 引擎将数据再次给到spiders
  9. spiders通过xpath解析数据，得到url或者数据
  10. spiders将数据或者url返回给引擎
  11. 引擎判断是数据还是url，是数据则交给管道处理，是url则交给调度器处理(即重复上诉逻辑)

日志#

一般推荐直接在settings文件中写入LOG_FILE = log.log来解决执行时调式显示过多。

import scrapy
from ..items import ScrapyDangdangItem

class DangSpider(scrapy.Spider):
    name = 'dang'
    # allowed_domains在多页下载中可以只写域名（category.dangdang.com）或者可以不写
    allowed_domains = ['category.dangdang.com']
    start_urls = ['http://category.dangdang.com/cp01.01.07.00.00.00.html']
    page = 1

    def parse(self, response, **kwargs):
        print("=============================")
        li_list = response.xpath('//ul[@id="component_59"]/li')
        for i in li_list:
            src_pic = i.xpath('.//img/@data-original').extract_first()
            name = i.xpath('./p[@name="title"]/a/@title').extract_first()
            price = i.xpath('./p[@class="price"]/span[1]/text()').extract_first()
            if src_pic:
                src_pic = src_pic
            else:
                src_pic = i.xpath('.//img/@src').extract_first()
            # print(src_pic, name, price)

            # 开始管道封装下载数据
            # 导入items的数据结构
            book = ScrapyDangdangItem(src_pic=src_pic, name=name, price=price)

            # 获取一个book就将一个book交给pipeline
            yield book

            if self.page < 100:
                self.page = self.page + 1
                more_url = f"http://category.dangdang.com/pg{self.page}-cp01.01.07.00.00.00.html"
                # scrapy.Request就是scrapy的get请求，还可以通过Request中的mete传递参数，
                # 并用response.meta['参数名']接受，callback就是要执行的那个函数
                yield scrapy.Request(url=more_url, callback=self.parse)

# useful for handling different item types with a single interface
from itemadapter import ItemAdapter
import requests
# 如果想要使用管道，就要在setting中开启管道
class ScrapyDangdangPipeline:
    # 该方法会在爬虫文件开始前就执行
    def open_spider(self, spider):
        self.f = open("book.json", "w", encoding="utf-8")

    def process_item(self, item, spider):   #item就是yield后面的book对象
        # 如果单单在这里使用文件的读写会使文件读写过于频繁
        # with open('book.json', 'a', encoding='utf-8') as f:
        #     f.write(str(item))

        self.f.write(str(item))
        return item
    # 该方法会在爬虫文件结束时执行
    def close_spider(self, spider):
        self.f.close()

# 多条管道的开启
# 1.首先创建一个管道类
# 2.在settings中开启管道
class DangDangDownloadPipeline:
    def process_item(self, item, spider):   #item就是yield后面的book对象
        url = "http:" + item.get("src_pic")
        name = "./books/" + item.get("name") + ".jpg"
        response = requests.get(url=url)
        with open(name, "wb") as f:
            f.write(response.content)

CrawlSpider-读书网#

  1. 继承自scrapy.Spider
  2. CrawlSpider可以自定义规则，再解析html内容的时候可以根据链接规则提取出指定的链接，然后载器向这些链接发送请求
  3. 提取链接
    1. allow = ()， 正则表达式，提取符合正则表达式的链接
    2. restrict_xpaths = (), 提取符合xpath规则的链接
    3. restrict_css = (), 提取符合选择器规则的链接

rules = ( Rule(LinkExtractor(allow=r'/book/1158_\d+.html'), callback=‘parse_item’, follow=False), ) 的 follwo如果为True则表示为是否按照链接提取规则一直跟进提取

使用步骤：

  1. 创建项目scrapy startproject 项目名
  2. 进入到spiders后执行scrapy genspider -t crawl 爬虫文件名 爬取链接

import scrapy
from scrapy.linkextractors import LinkExtractor
from scrapy.spiders import CrawlSpider, Rule
from ..items import ScrapyReadbookItem

class ReadSpider(CrawlSpider):
    name = 'read'
    allowed_domains = ['www.dushu.com']
    start_urls = ['https://www.dushu.com/book/1158_1.html']

    rules = (
        Rule(LinkExtractor(allow=r'/book/1158_\d+.html'), callback='parse_item', follow=False),
    )

    def parse_item(self, response):
        name = response.xpath('//div[@class="book-info"]/h3/a/@title').getall()
        book = ScrapyReadbookItem(name=name)
        yield book

class TestpostSpider(scrapy.Spider):

        url = 'https://fanyi.baidu.com/sug'

        data = {
            'kw':'apple'
        }
        yield scrapy.FormRequest(url=url, formdata=data, callback=self.parse_second)
