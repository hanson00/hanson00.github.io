哈希hash#

hash是一类算法，该算法接受传入的内容，经过运算后得到一串hash值

hash值的特点：

  1. 只要传入的内容一样得到的hash值必然是一样的
  2. 不能由hash反推内容
  3. 只要hash算法一样，无论校验的内容有多大得到的hash长度都是固定的

import hashlib

# 创建了一个hash工厂
hash_factory = hashlib.md5()

# 给工厂运送原材料
hash_factory.update("hello".encode('utf-8'))
hash_factory.update("world".encode('utf-8'))
res = hash_factory.hexdigest()  #helloworld
print(res)

hash_factory2 = hashlib.md5("helloworld".encode("utf-8"))
print(hash_factory2.hexdigest())
