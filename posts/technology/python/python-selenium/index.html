Selenium的等待#

Selenium的等待有以下三种：

  1. 强制等待time.sleep()等到网页加载完成再等待指定的秒数才能继续往下执行。
  2. 隐式等待，在指定等待时间到来前加载完了，直接继续往下执行，如果等待了指定的时间后还未成功打开网页就直接继续往下执行。
  3. 显示等待，最为灵活。两个人去玩，其中一人到了目的地，另一人告诉他不来了，他就一个人进去。

隐式等待判断网页是否加载完，但可能网页上某个JavaScript脚本加载速度特别慢，而JavaScript脚本只是个限制脚本，不影响网页的正常浏览，但仍然需要等待全部页面加载完成。

显示等待只需要判断某个特定元素是否加载出来就可以了。

隐式等待如下：隐式等待设置一次就会在整个程序中都起作用

from selenium import webdriver
from selenium.webdriver.support import expected_conditions
from selenium.webdriver.support.wait import WebDriverWait
from selenium.webdriver.common.by import By

url = 'https://www.csdn.net/'

chrome = webdriver.Chrome()
# 设置隐式等待，等待时间为隐式等待和显示等待时间长的那个
chrome.implicitly_wait(10)
chrome.get(url=url)

try:
    WebDriverWait(chrome, 20, 0.5).until(expected_conditions.presence_of_element_located((By.LINK_TEXT, u'博客')))
finally:
    print(chrome.find_element(By.LINK_TEXT, "博客").get_attribute('href'))
    chrome.quit()

WebDriverWait()可选择的参数有四个，一般情况下只用前三个，第一个参数是设置要打开的浏览器，第二个参数是设置超时时间，第三个是设置检查频率。

后面一部分的until(method,message="")，message返回的结果是0，即失败后要返回的消息。

向expected_conditions.presence_of_element_located((元组))

            target_name = target.find_element(By.XPATH, value="./ul/li[1]/a").get_attribute('title')
            target_price = target.find_element(By.XPATH, value="./ul/li[2]/span[1]/em").text
            target_address = target.find_element(By.XPATH, value="./ul/li[4]").text
            item["name"] = target_name
            item["price"] = target_price
            item["address"] = target_address
            self.save_data(item)
        self.next_page()

    def next_page(self):
        next_btn = self.chrome.find_element(By.XPATH, value="//a[@class='page_next_yes']")
        try:
            if next_btn:
                next_btn.click()
                time.sleep(0.1)
                self.son_page()
            else:
                self.chrome.quit()
        except Exception as e:
            print(e)

    def save_data(self, data):

        res = self.collection.insert_one(data)
        print("保存成功：",res)

    def run(self):
        self.visit_url()
