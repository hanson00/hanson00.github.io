Python-csv模块的学习#

引言：CSV (Comma Separated Values)逗号分隔值格式是电子表格和数据库的常见输入输出文件格式。

为什么会出现：由于当时不同程序读写数据时会出现细微的差别，这些多个来源的数据文件的格式差别让程序难以高效地处理数据

csv模块实现了 CSV 格式表单数据的读写，提供了“以兼容 Excel 的方式输出数据文件”或“读取 Excel 输出的数据文件”的功能。

csv模块的常见用法#

csv.reader()与csv.writer()#

csv.reader(csvfile, dialect=‘excel’, **fmtparams)

返回一个reader对象，该对象会逐行遍历csvfile。 csvfile可以是任何支持iterator协议的对象，并且每次调用的时候都会调用__next__方法返回字符串，文件对象和列表对象都适用。 如果csvfile是文件对象，则打开它时应该使用newline=""，对于后面两个参数可以见Python标准库有详细介绍

第二个参数dialect，其实代指的是dialect这一大类，

有如下可选参数： delimiter(中译:界定符)：用于分隔字段的单字符，默认为',' doublequote：当该属性为True时，双写引号字符。如果该属性为 Flase，则在 引号字符 的前面放置 转义符。默认值为 True quotechar：用于包住有特殊字符的字段，特殊字段有定界符,引号字符,换行符,默认为'"' ，如用引号分隔的字符串名"李 寻欢"，则在输出的时候用选中的符号代替引号 skipinitialspace：如果为True，则在输入错误的csv时抛出Error异常，默认为False

import csv
"""该csv文件内容如下
姓名,年龄,班级
温铭军,22,软件1194
二奶,18,物理
三奶,33,1198
"""
# csv.DictReader(f, fieldnames=None, restkey=None, restval=None, dialect='excel', *args, **kwds))
# f 指要操作的文件|| fieldnames 是所要指定的字典的键名，如果不给出则默认使用csv文件的第一行来作为键名\
# 当存储的数据没有列名的时候使用fieldnames对列名进行赋值
# 当存储的数据有列名的时候可以不使用fieldname
"""
使用fieldnames后输出为：
{'name': '姓名', 'age': '年龄', 'class': '班级'}
{'name': '雷军', 'age': '22', 'class': '软件1194'}
{'name': '雷二奶', 'age': '18', 'class': '物理'}
{'name': '雷三奶', 'age': '33', 'class': '1198'}
"""
with open("../1.CSV", ) as cs:
    result = csv.DictReader(cs, fieldnames=["name", "age", "class"])
    for row in result:
        print(row)


"""
不使用fieldname输出为：
{'姓名': '雷军', '年龄': '22', '班级': '软件1194'}
{'姓名': '雷二奶', '年龄': '18', '班级': '物理'}
{'姓名': '雷三奶', '年龄': '33', '班级': '1198'}
"""

csv.DictWriter()#

csv.DictWriter(f, fieldnames, restval='', extrasaction='raise', *dialect='excel', **args, ***kwds)

f为文件名

fieldnames是由键的组成的值作为序列传参

extrasaction 用于指定要执行的操作。 如果将其设置为默认值 ‘raise’，则会引发 ValueError 如果将其设置为 ‘ignore’，则字典中的其他键值将被忽略

（传入的字典的键不够时）如果传递给writerow()方法的键缺少fieldnames中的键时，默认会使用可选参数extrasaction写入，默认值 'raise'，则会引发 ValueError。 如果将其设置为 'ignore'，则字典中的其他键值将被忽略

（（没有指定fieldnames时）在缺少fieldnames参数时，默认会用restval用于指定值

import csv
"""该csv文件内容如下
姓名,年龄,班级
温铭军,22,软件1194
二奶,18,物理
三奶,33,1198
"""
# newline是每次写入完后，再次迭代写入时所要插入的值,默认是换行
with open("../1.CSV", "a", newline='') as cs:
    loadwriter = csv.DictWriter(cs,extrasaction="ignore", fieldnames=["姓名", "年龄", "班级"])
    loadwriter.writeheader()    #将参数fieldnames写入表头
    loadwriter.writerow({"姓名":"类四奶", "年龄":"16", "班级":"软件1194"})
    loadwriter.writerow({"姓名":"温铭军", "年龄":"22", "班级":"软件1194"})
print("csv.DictWriter写入完毕")
