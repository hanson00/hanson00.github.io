Cookie验证#

    • 带着Cookie再次请求最开始请求的网址
  • https://www.dsedt.gov.mo/zh_CN/pg_home

JavaScript的一些小记录#

  • 补环境，扣js最好使用var声明变量，变量放最上方

  • 使用var可以重复声明变量，并最新的变量覆盖旧变量，但let不行

1
2
3

var a = 20
var a = 10
console.log(a)
let b = 99
let b = 22
console.log(b)

  • let 声明的变量只在 let 命令所在的代码块内有效。

  • const 声明一个只读的常量，一旦声明，常量的值就不能改变。

总结：

  1. var 声明的变量属于函数作用域
  2. var 声明的变量存在提升（hoisting）（将变量提升到当前作用域的顶部）
  3. var 变量可以重复声明
  4. var在函数中声明的变量是属于局部变量
