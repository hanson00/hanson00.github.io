进程与线程#

每个主进程中至少都会有一个线程，当你创建一个新的线程时，就相当于有两个员工(线程)在工作具体的工作顺序由CPU决定，主线程会继续往下面执行，子线程也会自己执行自己的任务

在Python中一般不创建多进程，因为进程消耗的内存资源较大

t.start()，当前线程准备就绪，等待CPU调度 t.join()等待当前任务执行完成，当前任务执行完成之后才会往后面执行

import requests
from concurrent.futures import ThreadPoolExecutor
import csv
url = "http://www.xinfadi.com.cn/getPriceData.html"
header = {
"User-Agent": "Mozilla / 5.0(Windows NT 10.0;Win64;x64) AppleWebKit / 537.36(KHTML, like Gecko) Chrome / 94.0.4606.71Safari / 537.36"
}

def getdata(data):
    url = "http://www.xinfadi.com.cn/getPriceData.html"
    response = requests.post(url, headers=header, data=data)
    result = response.json()
    ts = result["list"]
    with open("北京新发地农产品数据.csv", "a", newline='') as f:
        csvwriter = csv.DictWriter(f, fieldnames=['prodName', 'lowPrice', 'highPrice', 'avgPrice', 'place', 'unitInfo',
                                                  'pubDate'],
                                   extrasaction='ignore')
        csvwriter.writeheader()
        for i in range(len(ts)):
            csvwriter.writerow(ts[i])


if __name__ == '__main__':
    with ThreadPoolExecutor(20) as t:
        for i in range(1, 60):
            data = {
                'limit': '',
                'current': f'{i}',
                'pubDateStartTime': '',
                'pubDateEndTime': '',
                'prodPcatid': '',
                'prodCatid': '',
                'prodName': ''
            }
            t.submit(getdata, data)
    print("OVER")

协程(多任务异步协程)#

协程是在单线程的基础上操作（多线程协作可以充分地利用CPU）

切换条件：当程序遇到IO操作时可以选择切换到其他任务(计算类型的任务)上

在微观上一个任务一个任务进行切换，切换条件是遇到IO操作(单线程的多个任务来回切换运行)

在宏观上就是多个任务共同运行

会使程序陷入阻塞状态的有：IO操作和requests.get()在网络请求返回前都会处于阻塞状态

协程分配的任务是由人来调度，线程池是由系统来调度的

异步任务和同步任务看是否需要等待某一操作

在编写协程代码时要注意：await(挂起)要写到async的函数里面，放在协程对象里面

#协程的编写
import time
import asyncio
async def func1():   #async:将该函数定义成异步协程函数
    print("我爱我们")
    # time.sleep(3)   #当前线程处于阻塞状态，CPU不为我工作,time.sleep()是串形同步操作，使得异步中断了
    await asyncio.sleep(3)
    print("这是第二次我爱你")

async def func2():   #async:将该函数定义成异步协程函数
    print("我爱22")
    # time.sleep(3)   #当前线程处于阻塞状态，CPU不为我工作
    await asyncio.sleep(3)
    print("这是第二次我爱22")

async def func3():   #async:将该函数定义成异步协程函数
    print("我爱33")
    # time.sleep(3)   #当前线程处于阻塞状态，CPU不为我工作
    await asyncio.sleep(4)  #改成这个异步操作代码就可以实现协程的异步处理
    print("这是第二次我33")

if __name__ == '__main__':

    #input()也会让CPU处于阻塞状态
    #requests.get()也会使程序处于阻塞状态
    #当程序处于IO操作时程序会陷入阻塞状态
    #协程：当程序遇到IO操作时，可以选择性的切换到其他任务
      #如果执行函数就会得到一个协程对象
      #协程函数的运行需要asyncio模块的支持
    t1 = time.time()
    f1 = func1()
    f2 = func2()
    f3 = func3()
    task = [f1, f2, f3]
    #一次性启动多个任务（协程）
    #多任务同时启动就需要asyncio.wait,把多个任务交给asyncio.wait(),而启动就需要asyncio.run()
    asyncio.run(asyncio.wait(task))
    t2 = time.time()
    print(t2-t1)

上面代码是同步操作的协程

import asyncio
import time


async def func1():
    print("我是第一")
    await asyncio.sleep(3)
    print("我是第一")

async def func2():
    print("我是第2")
    await asyncio.sleep(2)
    print("我是第2")

async def func3():
    print("我是第3")
    await asyncio.sleep(4)
    print("我是第3")

async def main():
    task = []
    task = [asyncio.create_task(func1()),
            asyncio.create_task(func2()),
            asyncio.create_task(func3())]
    await asyncio.wait(task)

if __name__ == '__main__':
    t1 = time.time()
    asyncio.run(main())
    t2 = time.time()
    print(t2-t1)
    #output:我是第一

# 我是第3

#aiohttp异步爬取优美图库
# requests.get()该操作是同步操作，如果想要
# 使用异步操作的话，就得使用aiohttp
import asyncio
import aiohttp
import time

urls = [
    "http://kr.shanghai-jiuxin.com/file/mm/20211130/zkc0inje5x0.jpg",
    "http://kr.shanghai-jiuxin.com/file/mm/20211130/ufmo0xczdbg.jpg",
    "http://kr.shanghai-jiuxin.com/file/mm/20211130/15dujfems5v.jpg"
]

async def downsave(url):
    name = url.split("/")[-1]
    async with aiohttp.ClientSession() as sessions:  # 这部的操作就相当于   <==> requests
        async with sessions.get(url) as s:
            with open(f"{name}.jpg", "wb") as f:
                f.write(await s.content.read())
    print("OVER")

async def main():
    task = []
    for url in urls:
        task.append(downsave(url))
    await asyncio.wait(task)

if __name__ == '__main__':
    t1 = time.time()
    asyncio.run(main())
    t2 = time.time()
    print(t2-t1)
