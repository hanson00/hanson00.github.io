json模块#

JSON（JavaScript Object Notation）是一种轻量级的数据交换格式。下面给出json模块的常用方法

  1. json.dump()：将Python数据对象以JSON格式数据流写入文件
  2. json.load()：解析包含JSON数据的文件为Python对象
  3. json.dumps()：将Python数据对象转换成JSON格式的字符串，结果是JSON格式的字符串
  4. json.loads()：将包含JSON的字符串、字节以及字节数组解析成为Python对象

生成的json字符串一定是双引号引导的。

Python中的json模块的使用方法

方法 功能
json.dump(obj,fp) 将Python数据类型转换并保存到json格式的文件里
json.dumps(s) 将Python数据类型转换成json格式的字符串
json.load(fp) 将json格式的文件读取数据并转换成Python的类型
json.loads(s) 将json格式的字符串转换成Python类型的字符串

import json
person = {"name": "hanson", "age": 18, "tel":[123, 321]}
print(type(person))
print(person)

# 将字典转换成json格式的字符串,intent指定缩进，不指定也行
j = json.dumps(person, indent=2)
print(type(j))  # <class 'str'>
print(j)

with open("../saves/data.json", "w") as f:
    json.dump(person, f)

import json
person = {"name": "杰克", "age": 18, "tel":[123, 321]}

# 将字典转换成json格式的字符串,intent指定缩进，不指定也行,ensure_ascii显示中文
j = json.dumps(person, indent=2, ensure_ascii=False)
print(j)
# 将json string 转换 python对象
pythonObj = json.loads(j)
print(type(pythonObj))  # output: <class 'dict'>
print(pythonObj)

import json
dic = {
    "name" : "饿饿饿",
    "age":[
        {
            "shuah":"sodjsijd"
        }
    ]
}


with open("22.txt", "w") as f:
    json.dump(dic, f)   # 将上述json数据写入文件

with open("22.txt", "r") as f:
    print(f.read())     # output:{"name": "\u997f\u997f\u997f", "age": [{"shuah": "sodjsijd"}]}
print("--------------------------------")
with open("22.txt", "r") as f:
    print(json.load(f))     # output:{'name': '饿饿饿', 'age': [{'shuah': 'sodjsijd'}]}

print("OVER")
