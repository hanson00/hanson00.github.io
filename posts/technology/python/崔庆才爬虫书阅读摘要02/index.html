对爬虫书的第三章摘录#

xpath#

etree.parse()解析本地文件 etree.HTML()解析服务器响应文件

xpath基本语法：

  1. 路径查询

    1. //：查找所有子孙节点，不考虑层级关系
    2. / ：找直接子节点

  2. 谓词查询 //div[@id] //div[@id="maincontent"]

  3. 属性查询 //@class

  4. 模糊查询 //div[contains(@id, "he")] //div[starts‐with(@id, "he")]

  5. 内容查询 //div/h1/text()

  6. 逻辑运算 //div[@id="head" and @class="s_down"] //title | //pric

注意：

  • 使用xpath help插件可能会出现解析前后不对的情况，因为在浏览器的解析，xpath解析的是渲染后网页可能与网页源代码有出入
  • 通过xpath获取的值的返回是一个列表

from lxml import etree

html = """
        <li><a href="http://www.baidu.com 666" id="baidu">百度1</a></li>
        <li><a href="http://www.baidu.com 666" id="baidu2">百度2</a></li>
        <li><a href="http://www.google.com" id="guge">谷歌</a></li>
"""

tree = etree.HTML(html)

alls = tree.xpath('//li/a[contains(@href, "666") and @id="baidu2"]/text()')
print(alls)	# ['百度2']

import requests
from lxml import etree


url = 'https://www.qqtxt.cc/sort2/1.html'

response = requests.get(url, verify=False)
response.encoding = 'gbk'
html = etree.HTML(response.text)
# position()>1 取位置序号大于1的元素，xpath序号从1开始
book_lists = html.xpath('/html/body/div[8]/ul/li[position()>1]')
# print(book_lists)
for item in book_lists:
    name = item.xpath('./span[1]/a/@title')[0]
    print(name)
response.close()

BeautifulSoup#

BeautifulSoup是Python的一个HTML或XML解析库，可以将输入的文档自动转换成Unicode编码，将输出的文档转换成UTF-8编码，除非文档没有指定的编码方式，不然无需指定编码方式。

对html 进行解析时，Beautiful Soup 支持解析器的选取，通常来说在选择解析器的时候需要记住两个点，一个是解析时间，另一个是兼容性。

Beautiful Soup 支持的解析器有四种：

html.parser，lxml，lxml-xml，html5lib。

这四种解析器的优点和缺点，文档中下面的表很容易看清：

在BeautifulSoup 4.0以前，大部分使用的是html.parser ，现在更多使用的是lxml。

from bs4 import BeautifulSoup

<html><head><title>The Dormouse's story</title></head>
<body>
<p class="title" id="这是第一个P节点id"><b>The Dormouse's story</b></p>

<p class="story">...</p>
"""

result = soup.find_all(name="a")
result2 = soup.find_all(attrs={"id":"link1"})

from pyquery import PyQuery

html = """
<ol>
<li><a href="XXXXXXX">的撒</a></li>
</ol>

<ul>
<li class="one-group" id="one"><a href="http://www.google.com">谷歌</a></li>

<li class="one-group"><a href="http://www.baidu.com">百度</a></li>
<li class="two-group"><a href="http://www.tenxun.com">腾讯</a></li>
<li class="two-group"><a href="http://www.hanson.com">hanson</a></li>

</ul>
"""

# 创建了一个PyQuery对象
p = PyQuery(html)

p_obj = p("ol li a")
# 上面的语句和注释的语句相同 p_obj = p("ol")("li")("a")
print(p_obj)
print(type(p_obj))  # output:<class 'pyquery.pyquery.PyQuery'>

id_obj = p("#one")
print("id_obj ", id_obj)

one_group = p(".one-group")
print("one_group ", one_group)
print(type(one_group))


# 获取所有符合条件的对象的文本
text = p("ul li a").text()
print(text)
print(type(text))

# 获取单个（第一个）对象的属性
text = p("ul li a").attr("href")
print(text)
print(type(text))

# 获取所有符合条件的属性值
text = p("ul li a").items()
for i in text:
    href = i.attr("href")
    text = i.text()
    print(text, href)

from pyquery import PyQuery

html = """
<ol>
<li class="FF"><a href="XXXXXXX">的撒</a></li>
</ol>

"""

# 创建了一个PyQuery对象
p = PyQuery(html)

# 在对象后面添加新标签
ol_a = p("ol li").after("""<li class="RR"><a href="WWWWWWWW">6666666</a></li>""")

# 在对象里面再插入新的标签
ol_a_in = p("ol li").append("""<img href="Photo place">【图片】</img>""")

# 有属性则可以修改，没有属性可以新增
ol_a_attr = p(".FF a").attr("href", "HELLO")
ol_a_attr_new = p(".FF a").attr("id", "WORLD")

# 删除节点或者删除属性
del_a_id = p(".FF a").remove_attr("id")
del_a_in = p(".RR").remove()

print(p)

from parsel import Selector

# 创建Selector对象
html = '''
<html>
    <body>
        <div class="container">
            <a href="https://example.com">Example</a>
            <img src="image.jpg" alt="Image">
        </div>
    </body>
</html>
'''

selector = Selector(text=html)

# 使用CSS选择器提取属性值
link_href = selector.css('a::attr(href)').get()
image_src = selector.css('img::attr(src)').get()

print("Link Href:", link_href)
print("Image Src:", image_src)
