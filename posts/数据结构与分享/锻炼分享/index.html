锻炼#

怎么吃？#

体重（磅为单位）×11=基本热量需求[不吃东西不运动的热量需求]

蛋白质#

优质蛋白质

  1. 鱼类
  2. 蛋类
  3. 精瘦肉
  4. 漆皮肌肉

劣质蛋白质

  1. 牛腩
  2. 较肥的肉

淀粉（锻炼开始和结束可以和蛋白质一起吃）#

优质淀粉

  1. 黑麦面包
  2. 红薯
  3. 全谷物面包
  4. 麸皮食品

劣质淀粉

  1. 土豆
  2. 高糖谷类
  3. 白米饭
  4. 白面包
  5. 干果不要吃太多

饮料#

优质

  1. 花草茶
  2. 矿物质水
  3. 水
  4. 茶

健身周期计划#

想要塑造肌肉就要摄入大量的热量

健身-减脂周期：#

  1. 先进行6个月的锻炼，同时摄入更多的热量
  2. 减脂阶段：在锻炼的途中减脂，控制热量的摄入，全程减脂
  3. 在稍微增加热量，增加肌肉
  4. 在稍微减少热量，减脂
