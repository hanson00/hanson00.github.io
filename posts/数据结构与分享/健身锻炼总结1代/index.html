分化训练#

训练频率：每周练1-2次，聊胜于无。每周练3-5次，较为合适。每周练6次，极限频率

最好训练后摄入碳水和蛋白质，碳水尽量有氧运动后吃

肌肉的生长是通过：力学张力，肌肉破坏，代谢压力

超量恢复#

超量恢复指的是在适当的休息和恢复时间内，超越了以前的体能和肌肉水平。这种恢复会导致身体更强壮、更耐力和更有适应性。

超量恢复的关键是：

  1. 适当的休息：确保你的身体有足够的时间来恢复，包括充足的睡眠和休息日。
  2. 良好的营养：提供足够的营养物质，包括蛋白质、碳水化合物和脂肪，以满足恢复和生长的需要。
  3. 逐渐增加训练强度：不断增加训练负荷，以促进超量恢复。
  4. 应用适当的恢复策略，如按摩、拉伸、冷热水疗法等。

协同肌群#

常见的协同肌群示例：

  1. 背部和肱二头肌：在进行引体向上（俯卧撑）时，背部肌肉（如背阔肌）和肱二头肌协同工作，以提供上升和下降的力量。
  2. 腿部和臀部：在进行深蹲或举重时，大腿肌肉（股四头肌）和臀部肌肉（臀大肌）协同工作，以支撑身体并提供起立和下蹲的力量。
  3. 胸部和三角肌：在进行卧推运动时，胸大肌和三角肌（前三角肌、中三角肌、后三角肌）协同工作，以推动重量。
  4. 腹部和腰部：在进行仰卧起坐或旋转运动时，腹部肌肉（如腹直肌）和腰部肌肉协同工作，以稳定和控制身体的运动。
  5. 肩部和上臂肌肉：在进行上拉或提举动作时，肩部肌肉和上臂肌肉（如肱二头肌和肱三头肌）协同工作，以支持举重动作。
  6. 臀部和大腿后侧肌肉：在进行死lift（硬拉）运动时，臀部和大腿后侧肌肉（腘肌群）协同工作，以提供起重的力量。

小肌肉群

小肌肉群的部位：

  1. 前臂肌肉：包括屈腕肌和伸腕肌，位于前臂上，负责手腕的弯曲和伸展。
  2. 肩部肌肉：包括肩部旋转肌（肩袖肌群）和肩部稳定肌肉，负责肩关节的稳定和旋转。
  3. 颈部肌肉：包括颈部屈肌和伸肌，用于支撑头部和进行颈部的弯曲和伸展动作。
  4. 核心肌肉：包括腹直肌、腹外斜肌、腹内斜肌和腹横肌，负责维持身体的稳定性和支持脊柱。
  5. 髋部肌肉：包括髋外转肌、髋内转肌和髋屈肌，用于支持髋关节的稳定性和动作。
  6. 膝盖周围肌肉：包括膝关节的屈肌和伸肌，用于支持膝盖的运动和稳定性。
  7. 手部肌肉：包括手指屈肌和手指伸肌，用于手部的弯曲和伸展动作。

小肌肉群休息：

  1. 日常休息：小肌肉群通常需要每天休息。这意味着不要在连续的天数内重复相同的小肌肉群的训练，以允许肌肉有时间恢复和生长。
  2. 48小时原则：大多数人遵循48小时的原则，这意味着在锻炼小肌肉群后，至少应该等待48小时才能再次锻炼同一部分肌肉。这是因为肌肉需要时间来修复和增长。
  3. 合理分配：合理分配锻炼小肌肉群，以确保不会过度疲劳或导致过度训练。
  4. 疼痛和不适：如果你在锻炼后感到明显的疼痛、不适或肌肉酸痛，那是身体在告诉你需要更多的休息。不要急于再次锻炼同一肌肉群。

大肌肉群

大肌肉群的部位：

  1. 胸大肌（Pectoralis Major）：位于胸部，用于上肢的前部移动，如推举和压力运动。
  2. 背阔肌（Latissimus Dorsi）：位于背部，用于向内旋转和拉伸上肢，如引体向上和划船。
  3. 腿部大肌肉群：
    • 股四头肌（Quadriceps）：位于大腿前部，用于大腿的伸展动作，如蹲举和深蹲。
    • 腘肌群（Hamstrings）：位于大腿后部，用于大腿的弯曲动作，如弯腰和坐姿腿屈。
  4. 肩部大肌肉群：
    • 三角肌（Deltoids）：位于肩部，用于肩部的抬举和旋转动作，如推举和侧平举。
  5. 上臂肌肉群：
    • 肱二头肌（Biceps Brachii）：位于上臂前部，用于肘部的屈曲动作，如弯举。
    • 肱三头肌（Triceps Brachii）：位于上臂后部，用于肘部的伸展动作，如推胳膊。

大肌肉群休息：48-72小时
