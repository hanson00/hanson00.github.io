问题：关于Scrapy框架#

问题： Signature of method ‘QiubaiSpider.parse()’ does not match signature of base method in class ‘Spider’

问题描述： 方法 ‘BaiduSpider.parse()’ 的签名与类 ‘Spider’ 中基方法的签名不匹配

问题状态： 已解决

解决方法： 将Scrapy生成的parse()方法的参数后面添加**kwargs即可 即将方法修改为

def parse(self, response, **kwargs):
    pass

问题：关于selenium元素获取过期或被刷新#

问题报错：selenium.common.exceptions.StaleElementReferenceException: Message: stale element reference: element is not attached to the page document

问题描述：过期元素引用异常过期元素引用:元素没有附加到页面文档

问题状态： 已解决

报错原因：因为所打开的页面进行了刷新或者跳转而导致的元素定位获取失败

解决方法：

  1. 重新使用完整的路径定位（在获取了大范围需要在该大范围中在细化查找时）
  2. 等待一段时间后再进行元素定位

问题：关于爬虫表单数据的提交报错#

问题报错：{“timestamp”:1669252234132,“status”:415,“error”:“Unsupported Media Type”,“message”:“Content type ‘application/octet-stream’ not supported”,“path”:"/wechat/phonePwdLogin"}

问题描述：“不支持的媒体类型”，“message”：“不支持内容类型”applicationoctet stream“

问题状态： 已解决

报错原因：在遇到一些POST是payload报文的时候（该报文是一种json格式的报文），传入的对象应该是JSON格式的。

在Python中JSON格式其实也就是所字典对象引用的符号是双引号而不是单引号。

JSON格式的字符串和str得到的字符串是不同的。JSON格式的字符串除了要满足字典类型的格式外，所有的字符串对象必须是双引号。

d_d = {"a": '1', "b": 2}

js = json.dumps(d)
js_d = json.dumps(d_d)
print("str格式的样子：",string, string_d, string == string_d)

print("JSON格式的样子：",js, js_d, js == js_d)

print(string == js)

requests.post()#

requests.post()进行post请求时，传入的报文参数有两个：data和json，常见的form表单可以使用data参数进行提交，data对象是python中的字典类型，但在上述报错中是因为遇到了payload这种报文，该报文是一种json格式的报文，所以传入的对象也应该是JSON格式。

下面给出两种不同的表单类型

解决方法：

  1. 在POST提交的data中将data参数转换成JSON格式
1

requests.post(url, data=json.dumps(data))
  1. 直接使用json提交数据
1

requests.post(url, json=data

仍需注意的问题：

  • 如果参数为JSON数据，可以直接传入json参数，它将自动编码并将Content-Type的置为application/json

  • 如果data传递的参数为字符串，如：json.dumps(data)，则request对参数进行url编码，Content-Type的值为None，所以data传字符串时，一定要在header中指定Content-Type

1
2
3
4

data = {表单数据}
headers={"Content-Type": "application/json"}
response = requests.post("https://httpbin.org/post", headers=headers,data=json.dumps(data))
print(response.text)
  • 如果data传递的是字典、元组组成的列表或列表作为值的字典，则request对参数进行url编码，Content-Type的值为application/x-www-form-urlencoded

json = data 传的是json格式数据给后台

data = data 传的是字典结构给后台，原生的form表单
