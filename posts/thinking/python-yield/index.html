Python-yield#

对于Python的yield的一些用法的示例

示例一：#

yield遇到时会产生一个迭代器（生成器）对象，并会有记忆的记录当前位置以及变量值

def dog(name):
    print(f"狗哥{name}准备开始吃东西")
    while True:
        x = yield
        print(f"狗哥{name}开始吃东西{x}")

if __name__ == '__main__':
    g = dog('alex')
    next(g)
    g.send('骨头')
    g.send('香肠')
# 输出如下：
# 狗哥alex准备开始吃东西
# 狗哥alex开始吃东西骨头
# 狗哥alex开始吃东西香肠

next(g)相当于《====》g.send(None)，可以理解为负责初始化一个yield对象。

def dog(name):
    food_list = []
    print(f"狗哥{name}准备开始吃东西")
    while True:
        # x是拿到yield接收到的值，返回的是food_list
        x = yield food_list
        print(f"狗哥{name}开始吃东西{x}")
        food_list.append(x)

if __name__ == '__main__':
    g = dog('alex')
    next(g)
    res = g.send('骨头')
    print(res)
    res = g.send('香肠')
    print(res)
# 输出如下：
# 狗哥alex准备开始吃东西
# 狗哥alex开始吃东西骨头
# ['骨头']
# 狗哥alex开始吃东西香肠
# ['骨头', '香肠']
