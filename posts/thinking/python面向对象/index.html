Python类的封装继承和多态的记录#

对象可以理解为是“容器”，一个盛放数据和功能的一个容器||整合功能

类也是“容器”可以存放同类对象共有的数据和功能

def __init__(self):

  1. 会在调用类时自动触发执行，用来初始化对象自己独有的数据
  2. 该方法本意是用来初始化对象的属性，但是如果穿插其他任意代码也会执行
  3. 该方法的返回值必须是None

将封装的属性进行隐藏操作，但Python并没有真正的私有（隐藏）如果想要访问其实是可以访问到的通过_类名__隐藏名可以访问。并且这种隐藏对外不对内，类里面可以访问，且只在类体的隐藏才会发生变形

    year = 2022

    def __f1(self):

继承#

继承表达的是一种什么是什么的关系

通过继承可以解决类与类的代码冗余问题 继承可以丰富子类的属性和功能

  • Python3的类如果没有继承别的类，那么会默认继承Object类
  • Python支持多继承但不太建议使用多继承，可能会随着继承父类的增多会使可读性变差。
  • 可以使用Mixins的规范使用多继承

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def f1(self):
        print("Person f1")

    def f2(self):
        print("Person f2")
        # 如果想访问Person对象下面的f1可以使用Person.f1()
        self.f1()

class Students(Person):
    def __init__(self, name, age, sid):
        super().__init__(name, age)
        self.sid = sid
    def f1(self):
        print("Students f1")


student1 = Students("jhon", 12, "202201")
student1.f2()

例如：顶级父类交通工具类，下面有喷气式飞机类和直升飞机类、汽车类等三个子类，两个飞机有一个共同的方法飞可以提取到一个class FlyMixin:类中，用类名+Mixin来更加直接明了的表明是功能类
